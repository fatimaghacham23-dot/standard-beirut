# Standard

Independent Next.js landing page for Standard, using the premium dark/noir coffee template.

## Local commands

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run build
```

## Vercel deploy note

Import this folder as its own Vercel project. Use `npm install` for installation and `npm run build` as the build command. Vercel will produce and serve the Next.js output from `.next` during deployment.
